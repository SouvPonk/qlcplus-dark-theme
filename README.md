# QLC+ blue-ish dark theme 
Simple dark theme for QLC+. 

Fork from https://github.com/EmerickH/qlcplus-dark-theme  and https://docs.qlcplus.org/v4/advanced/custom-ui-style

Copy **qlcplusStyle.qss** to:

- Linux    **/home/YOURUSERNAME/.qlcplus/qlcplusStyle.qss**
- Windows  **C:\Users\YOURUSERNAME\QLC+\qlcplusStyle.qss**
- MacOS    **~/Library/Application Support/QLC+/qlcplusStyle.qss**
